# Rockwell Agency – Contractor Theme

WordPress theme for home service contractor sites (HVAC, plumbing, electrical, roofing, cleaning). Built for lead generation and fast performance.

## What’s in this repo

- **contractor-theme/** – The WordPress theme (upload as ZIP or copy to `wp-content/themes/`)
- **create-theme-zip.ps1** – PowerShell script to build an installable `contractor-theme.zip`

## Quick start

1. Build the ZIP: run `create-theme-zip.ps1` (or zip the `contractor-theme` folder with that name as the root).
2. In WordPress: **Appearance → Themes → Add New → Upload Theme** and choose the ZIP.
3. Activate **Contractor Pro** and set **Settings → Reading** to a static front page.
4. Customize: **Appearance → Customize → Contact & CTA** (phone, hero text, hero image).

See **contractor-theme/README.md** for full theme docs.
